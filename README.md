# OpenPortfolio

**Micheal Kinney (aliasfoxkde) - Developer Portfolio**

A world-class portfolio website consolidating all projects, resources, and professional presence into one cohesive, polished, and interactive experience.

## 🚀 Live

- **Portfolio:** https://portfolio.cyopsys.com (or Cloudflare Pages URL)
- **GitHub:** https://github.com/aliasfoxkde
- **Projects:** See projects section for all deployed sites

## 🛠️ Tech Stack

- **Framework:** Vite + React 19
- **Language:** TypeScript (strict mode)
- **Styling:** TailwindCSS v4
- **UI Components:** Radix UI + Lucide Icons
- **Animation:** Framer Motion
- **Testing:** Vitest + Playwright
- **Deployment:** Cloudflare Pages

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run tests
npm test

# Run e2e tests
npm run test:e2e

# Lint
npm run lint
```
